settings - Collaborators #

第一次提交 #

  • 组长把代码写好提交到自己的仓库上
  • 组长需要发送pull request 组长请求老师合并
  • 因为组长有老师的权限 所以组长可以自己合并

  • 组长在刚才的fork里添加自己的组员

    现在组长可以操作老师的项目，但是组长和老师的项目不是一个，刚才组长已经加组员了 现在组员和组长用的是同一个仓库

  • 解决冲突>>>>> ======= <<<<<<<< 再进行git add git commit 就可以解决冲突了

  • git reset --hard 版本号 回退到哪个版本
